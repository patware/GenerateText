GenerateText - A simple utility that generates text for QA

The idea behind this application is to allow QA/Testers to generate text with known lengths and patterns. The QA/Tester simply copies the generated texts into the application to test's input boxes.

The "Approved by Annette" feature is a feature for a good friend a mine, Annette Bose. The goal was to add a trailing/suffix that indicates the end of the string, but with its actual length. This way, if the string being tested gets truncated down the line, the trailing/suffix would become mangled.

Ex: input string: abcdef_10_ the input box has 10 characters, but the database has 8, the return string would become abcdef_1 so a bug.

Extremely simple.

Here are features that I'd like to add:

  • The ability to change the suffix character "_"
  • A "snoop like" utility that would "paste" the right number of characters based on the input boxe's maxlength. The user would simply hit paste, paste, paste (or whatever CTRL... combo) and the utility would generate the targeted max length.
